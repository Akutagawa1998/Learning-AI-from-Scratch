# Day 2 - MNIST 数据管线说明

## Shape 语义
- 默认输出：`X` 为 `(B,1,28,28)`，`B` 是 batch 大小，`1` 是灰度通道。
- flatten 模式：`X` 为 `(B,784)`，等价于把 `(1,28,28)` 展平成一维向量，适合 MLP。
- `y` 为 `(B,)`，每个元素是 0-9 的类别标签。

## 归一化策略
- `normalize="none"`: `ToTensor()` 把像素从 `[0,255]` 转成 `[0,1]` 的 `float32`。
- `normalize="mnist"`: 先做 `[0,1]`，再用 `mean=0.1307`、`std=0.3081` 标准化。
  - 标准化后数值不再局限于 `[0,1]`，而是以 0 为中心分布。
